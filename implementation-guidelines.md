# Implementation Guidelines
SNAB is a fairly loose format, and can be adjusted to fit the needs of any usecase. However, for an implementation to be SNAB complient, they should follow these guidelines.

- The core feature-set for any target version of SNAB must be included.
- The file structure must follow the SNAB specification.
- Implementations should include a way to add user-definable datatypes to fit a programmers specific needs.
