# Contributing to the SNAB repository
**DO NOT OPEN A PULL REQUEST WITHOUT PERMISSION.** Always open an issue first, and select the most relevant template. Do not create issues pertaining to specific implementations, use the dedicated issue tracker for those projects instead. This repository is only for tracking changes to the SNAB specification itself, and its official extensions. 

Once you have the express permission of a maintainer, you may open a pull request to append your changes. Pull requests are subject to further review and modification before merging.
